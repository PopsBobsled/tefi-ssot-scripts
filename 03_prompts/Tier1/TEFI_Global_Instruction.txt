WORKFLOW 1 — ACTION PROMPT (use when told "Process CV")

Trigger phrase: "Process CV: [filename]"

Steps:
1. Read the named PDF from TEFI_Tier1_CV_Input
2. Extract all text from the PDF
3. Read TEFI_Tier1_00_Shared_Standards.md
4. Read TEFI_Tier1_01_Action_Prompt.md
5. Apply the Action Prompt to the extracted CV text
6. Extract the client email address from the CV text.

   please add manually" at the top of the output file.
7. Save the output as a .txt file in TEFI_Tier1_Output
   named exactly: [Candidate_Lastname]_W1_action_email.txt
8. Report back: candidate name, email address found, 
   output filename, and word count of the email generated.

WORKFLOW 2 — FINISHING PROMPT (use when told "Process Report")

Trigger phrase: "Process Report: [CV filename] + [Reply filename]"

Steps:
1. Read the named CV PDF from TEFI_Tier1_CV_Input
2. Read the named reply .txt file from TEFI_Tier1_CV_Input
3. Extract all text from both files
4. Read TEFI_Tier1_00_Shared_Standards.md
5. Read TEFI_Tier1_02_Finishing_Prompt.md
6. Apply the Finishing Prompt using both inputs
7. Extract the client email address from the CV text.
   If no email is found, write "EMAIL NOT FOUND — 
   please add manually" at the top of the output file.
8. Save the output as a .txt file in TEFI_Tier1_Output
   named exactly: [Candidate_Lastname]_W2_tier1_report.txt
9. Report back: candidate name, email address found, 
   output filename, word count, and which sections 
   received a "To strengthen this section:" flag.
